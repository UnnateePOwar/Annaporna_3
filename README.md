This project is a static, beginner-friendly web app that stores data in localStorage.
Features:
- Local authentication (register/login) using SHA-256 stored in localStorage
- Request creation with client-side geocoding (OpenStreetMap Nominatim)
- Local area map on home (centers on user location when permitted)
- Simple animations and transitions
To run:
Open index.html in a browser (or deploy to GitHub Pages / Netlify).
